## Requirements
- [PHP 7.2](https://php.net/)
- [Apache](https://httpd.apache.org/) or [nginx](https://nginx.org/)
- [composer](https://getcomposer.org)
- [node.js](https://nodejs.org)
- [npm](https://www.npmjs.com/)
